This package contains a self-contained frontend (pure static) designed to run from GitHub Pages or any static host.

Files included:
- index.html
- styles.css
- app.js
- industry-list.json
- kpi-list.json
- i18n_th.json, i18n_en.json
- manifest.txt
- runbook_frontend.txt
- README.txt
- logo.png (placeholder)

Usage:
1) Unzip and inspect files.
2) Upload the folder contents to your GitHub repo root or GitHub Pages folder.
3) Open index.html in browser (or deploy via GitHub Pages).

This frontend is intentionally static; no backend or external APIs are used.
